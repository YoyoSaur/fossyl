Get up and running with fossyl in minutes.

## Installation

Install fossyl using your preferred package manager:

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install fossyl
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add fossyl
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add fossyl
    ```
  </TabItem>
</Tabs>
